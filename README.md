🌿 Mental Wellness App Prototype
📌 Overview

This project is a Figma-based prototype of a mental wellness application designed to help users manage stress, relax their minds, and improve emotional well-being. The app combines interactive features like an AI assistant, calming games, and guided breathing exercises to create a safe and supportive digital space.

🎯 Purpose

The main goal of this prototype is to:

Help users reduce stress and anxiety
Provide a safe space to express thoughts and emotions
Encourage healthy mental habits through simple daily activities
Make mental wellness engaging and accessible
🚀 Key Features
🤖 Cleo AI (AI Assistant)
Users can chat with Cleo AI to express their thoughts and feelings
Provides supportive and calming responses
Helps users feel heard and understood
🌬️ Breathing Exercises
Guided breathing techniques to help users relax
Useful during anxiety, stress, or panic moments
Promotes mindfulness and calmness
🎮 Relaxing Games
Simple, fun games designed to reduce stress
Helps distract users from negative thoughts
Improves mood and mental refreshment
🎨 User Interface
Aesthetic dark mode with calming colors
Clean and easy-to-use design
Smooth navigation between features
⚙️ How It Works
The user opens the app and lands on the home screen
They can choose between different features:
Talk to Cleo AI
Try breathing exercises
Play relaxing games
Each feature is designed to help the user relax in a different way:
AI → emotional support
Breathing → physical calmness
Games → mental distraction
The user can switch between features anytime based on their needs
🧪 User Testing & Feedback

The prototype was tested using multiple user personas. Users reported that:

Cleo AI helped them feel emotionally supported
Breathing exercises reduced stress quickly
Games helped improve mood and reduce overthinking
The app felt like a safe and calming space
💡 Conclusion

This prototype demonstrates how technology can be used to support mental wellness in a simple and engaging way. By combining AI interaction, mindfulness techniques, and interactive elements, the app creates a balanced approach to stress management.
